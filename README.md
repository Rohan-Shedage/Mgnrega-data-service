# MGNREGA Data Service

This project is a full-stack web application that manages and tracks data related to MGNREGA (Mahatma Gandhi National Rural Employment Guarantee Act). It includes a Node.js + Express backend connected to a MySQL database, providing APIs for worker management, project records, and daily work tracking. The frontend allows admins to add, view, and manage MGNREGA data efficiently.
